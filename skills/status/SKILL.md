---
description: Show Adline client status, earnings balance, link state, and version. Use when the user asks about their Adline earnings, whether the ad line is on, or whether the client is up to date.
---

Run `adline status` and summarize the output for the user: client version, paused or active, linked account (if any), and current balance.

If `adline` is not on PATH, the client is not installed; point the user to `/adline:install`.
